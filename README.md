## Project Overview
A complete end-to-end Excel project where raw sales data was transformed into meaningful and actionable business insights. This dashboard helps analyze sales performance, profit trends, and customer contribution with clear visual storytelling.

This project focuses on using Excel to clean, organize, analyze, and visualize business data. The final dashboard provides a comprehensive view of sales and profit performance across multiple dimensions such as time, geography, and customers
